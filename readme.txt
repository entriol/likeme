It just makes your posts likeable.

== Description ==

I was asked to develop a Wordpress plugin to get approval in my intern. That's motivation behind this plugin.

== Features ==

* Like button to your posts.
* Widget to show most upvoted posts.
* Admin page to show most liked tags in your posts.
